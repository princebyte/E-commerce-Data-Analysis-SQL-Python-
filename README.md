# E-commerce Data Analysis (SQL + Python)

## Project Overview
This project focuses on analyzing an e-commerce dataset to uncover business trends and customer insights. By integrating **SQL** for data extraction and **Python** for analysis, I have explored key performance indicators (KPIs) relevant to the retail industry.

##  Tech Stack
* **Language:** Python
* **Database:** MySQL
* **Libraries:** Pandas, Matplotlib, Seaborn, mysql-connector-python
* **Environment:** Jupyter Notebook

##  Key Insights Extracted
* **Customer Demographics:** Analyzed the distribution of unique customers across different cities and states.
* **Order Analysis:** Quantified the total volume of orders placed during the year 2017 to identify growth patterns.
* **Sales Intelligence:** Utilized complex **SQL Joins** to calculate total revenue generated per product category.

##  How to Navigate
1. Open the `ecommerce.ipynb` file above to view the complete Python source code and SQL queries.
2. The notebook includes data cleaning, analysis, and insights derived from the database.
